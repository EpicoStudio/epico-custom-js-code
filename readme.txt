== Description ==

The Épico Custom Javascript Code plugin allows you to easily add custom JavaScript code to the head, body, and footer sections of your WordPress site. Whether you need to integrate analytics tracking, embed custom scripts, or make specific modifications, this plugin simplifies the process.

Main features:

- Insert custom JavaScript code in the head section.
- Place JavaScript code right after the opening of the body tag.
- Add JavaScript code right before the closing body tag.
- Option to remove code from the database upon plugin deactivation.

For more information, visit the [plugin homepage](https://epico.studio).

== Installation ==

1. Upload the `epico-custom-javascript-code` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to 'Settings' -> 'Epico Custom Javascript Code' in your WordPress dashboard.
